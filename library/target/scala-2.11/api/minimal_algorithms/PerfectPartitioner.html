case class PerfectPartitioner(numPartitions: Int, itemsCntByPartition: Int) extends Partitioner with Product with Serializable

Distributes data into numerically equal partitions thanks to object's unique key that represents ranking position.

numPartitions

Number of partitions.

itemsCntByPartition
